# 📊 Behavioral Segmentation Engine (RFM Analysis)

The **Behavioral Segmentation Engine** is a data analytics project that performs customer segmentation using **RFM (Recency, Frequency, Monetary) analysis**. The project processes raw e-commerce data using **SQL** and visualizes insights through an interactive **Power BI dashboard**.

It helps identify customer behavior patterns, detect churn risks, and support data-driven marketing and business decisions.

---

## 🚀 Key Features

- RFM-based customer segmentation  
- Data cleaning and transformation using SQL  
- Use of window functions (`NTILE`) for scoring  
- Customer persona classification (Champions, Loyal, Lost, etc.)  
- Revenue and customer behavior analysis  
- Interactive Power BI dashboard  

---

## 🛠️ Tech Stack

- **SQL (MySQL)** – Data processing and analysis  
- **Power BI** – Visualization and dashboard  
- **RFM Analysis** – Core segmentation technique  

---

## 📂 Dataset

- Olist Brazilian E-commerce Dataset (Kaggle)  

Includes:
- Customers Dataset  
- Orders Dataset  
- Order Payments Dataset  

---

## 📈 Insights Generated

- Customer distribution across segments  
- Revenue contribution by segment  
- Monthly revenue trends  
- One-time vs repeat customers  

---

## 🎯 Project Goal

To transform raw transactional data into meaningful customer insights and enable better business decision-making through behavioral segmentation.

## 📌 Outcome
